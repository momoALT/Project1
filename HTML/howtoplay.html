How to Play

The game character is the mouse cursor!

try to survive without getting hit once, if you get hit the game will end.

this game is made up of several timed survival minigames, not completing a minigame in time will end the game. However, completing a minigame in time will go into the next minigame.

occasionally, stars will spawn, there are 3 different type and each give a different amount of points (100, 250, 500) so try and collect as many as you can and rack up points without dying.



At the end of each completed minigame there is a chance that you can obtain rare items. the type of item you recieve can be told by the color: Red = Cursed Item, Gold = Blessed item.



Thats all you need to know to get started, enjoy!
